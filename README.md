# 💎 HTML-Tools-Public: AI駆動型データ解析・業務効率化ツール集

## 🚀 プロジェクト概要

このリポジトリは、**非エンジニアである私が、AIツールチェーン（AntiGravity, Codex, ChatGPT/Gemini）の力を最大限に活用して開発している、全てのウェブベースのプロトタイプおよびツール群**を格納しています。

専門性の高い課題（例: 生体信号解析、複雑なシフト管理）に対し、**「ロジック（Python）の正確性」**と**「UI（HTML/JS）の使いやすさ」**を両立させることを目指しています。

### 統合と方針について

本リポジトリは、2025年12月より、これまで個別に存在していた全てのHTML/JavaScriptツールを一つに統合しました。この統合により、AI開発のコンテキストを統一し、ツールの開発効率とコード品質の向上を図ります。

## 🎯 行動指針：なぜこのリポジトリが価値を持つのか

私の活動の核は、**「AIとの協調作業の過程を資産化すること」**です。

1.  **AI支援による問題解決の記録**:
    すべてのツールの開発過程は、Obsidianのデイリーログに詳細に記録されています。このログには、AIへのプロンプト、生成コードのエラー、そして私が手動で解決した**「最も価値ある知恵」**が含まれています。
2.  **実データへの挑戦**:
    KaggleやPhysioNetなどの公開データセットを題材にすることで、机上の空論ではない、実務レベルの解析ロジックを実装しています。
3.  **継続的な改善**:
    `main` ブランチは常に安定版です。すべての開発は `feature` ブランチで行い、常に安全性を確保しながらツールの改善を続けます。

## 🛠️ 開発環境と技術スタック

* **IDE/エディタ**: VS Code
* **AI開発支援**: AntiGravity Agent, GitHub Copilot, Codex, ChatGPT/Gemini
* **言語**: HTML5, CSS3, JavaScript (フロントエンド), Python / MATLAB (解析ロジック), R（統計解析）
